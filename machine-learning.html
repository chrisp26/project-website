Machine learning

Regression analysis

The goal is to generate a model which can be used to predict the popularity of a song. For this I have used scikit-learn to run the following regression models:

  1. Linear Regression
  2. Ridge Regression
  3. Lassoo Regression
  4. Elastic Net Regression

Evaluation

All regression models returned very low R-square scores. Even when removing variables and running with different combinations it became clear quickly that my original hypothesis was going to fail.

Initially when creating these models the were done with the sample of ~3,000 popularity measures. I spent time retrieving more data via the Spotify API to increase this sample to ~75,000. This only resulted in the lower R-square scores shown here.
